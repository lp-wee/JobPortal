# 🚀 JobPortal Hub

**JobPortal Hub** — это современная веб-платформа для поиска работы и подбора сотрудников.
Проект реализует удобное взаимодействие между соискателями и работодателями через интуитивный интерфейс и масштабируемую архитектуру.

---

## 📌 О проекте

Платформа предназначена для:

* 📄 поиска вакансий
* 🧑‍💼 размещения предложений о работе
* 🔗 взаимодействия между кандидатами и работодателями

Система поддерживает разные роли пользователей и предоставляет отдельные интерфейсы для каждой из них.

---

## 🧩 Архитектура проекта

Проект построен по принципу **SPA (Single Page Application)** с разделением на frontend и backend.

### 📁 Структура

```
src/
 ├── components/
 ├── pages/
 ├── features/
 ├── services/
 ├── hooks/
 ├── utils/
 └── types/

supabase/
 ├── migrations/
 ├── functions/
```

## ⚡ Установка и запуск

```bash
# Клонирование репозитория
git clone https://github.com/your-username/jobportal-hub.git

# Переход в папку проекта
cd jobportal-hub

# Установка зависимостей
npm install

# Запуск проекта
npm run dev
```

---

## 🔧 Переменные окружения

Создайте файл `.env` и добавьте:

```
VITE_SUPABASE_URL=your_url
VITE_SUPABASE_ANON_KEY=your_key
```
